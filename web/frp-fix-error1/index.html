frpc 客户端启动 lookup 错误

刘军   in 武汉
2022-02-25
  • frp

# 报错

frpc 客户端启动 login to server failed: dial tcp: lookup 错误

2022/02/25 18:20:40 [W] [service.go:105] login to server failed: dial tcp: lookup ddns.t3cc.cn on 218.104.111.122:53: read udp 192.168.33.77:56941->218.104.111.122:53: i/o timeout

# 解决

将frpc.ini 中的server_addr的域名改成ip地址（ping命令或获取）即可，server_addr = 157.0.98.78

server_addr = ddns.t3cc.cn // 例如我对外访问的网址实际的IP是：157.0.98.78
server_port = 7000

# 启动命令

./frpc -c ./frpc.ini
